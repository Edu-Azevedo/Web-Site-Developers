Sobre a Biblioteca fullPage.js

Design

Com essa biblioteca você consegue um layout rápido e bonito onde não é dificil fazer um site One Page, agora basta usar essa biblioteca e você tera uma linda estrutura de site One Page!

Código

A biblioteca fornece uma ampla opções para construir seu site, ela é bem pratica de usar e fácil tem um documento no site da biblioteca caso você tenha dúvidas.

Desenvolvimento

O tempo que você vai poupar usando essa biblioteca é brincadeira! usando ela você só precisa se preocupar em deixar o site do seu jeitinho sem perder tempo montando uma estruta de site One Page.

Como Usar?

Para aplicar a biblioteca você pode pegar tanto os link para CND quanto baixar os arquivos da biblioteca (quando for pegar os arquivos pegue o fullPage.css e fullPage.js ou ate mesmo a versão minificada fullPage.min.js), você consegue eles no site da biblioteca. Veja como aplicar abaixo!

Criando as Telas!

Após você colocar os links igual o passo a passo atrás, basta agora criar as telas com html, vamos criar uma div onde ela vai ter um id que vai ser fullpage (obrigatorio ter esse id para usar a biblioteca) não se preocupe porque e bem fácil de usar a biblioteca.

Usando a Biblioteca!

Após de você criar as telas com o html, vamos criar um script para aplicar de fato a biblioteca ela tem diversos comandos para você deixar seu site One Page do jeito que quiser mas vamos usar um para pelo menos ter uma cor cada tela, a diversos códigos mas vou ensinar o básico. Veja como fazer o script abaixo.

Esta Pronto!

Agora a estrutura do seu site One Page está pronta, agora vem de você costumisar do jeito que quiser, não esqueça de ir para o site da biblioteca ver como usar em documentos e muito mais!
